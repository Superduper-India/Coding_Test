# 음양 더하기
## 1. 문제
***
### 원하는 것
a,b의 내적을 return하는 함수를 완성하라.

### 조건
a,b 1차원 정수배열은 길이가 같다.
a,b의 내적은 a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1]이다.
(n은 a,b의 길이)

## 2. 계획
***
a,b 배열의 길이가 같은 것을 이용한다.
길이만큼 for문을 돌려서 answer += a[i]*b[i] 되게 한다.

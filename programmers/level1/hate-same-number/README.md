# 같은 숫자는 싫어
## 1. 문제
***
### 원하는 것
배열 arr에서 연속적으로 나타나는 숫자는 제거하고 남은 수들을 return 하는 solution 함수를 완성

### 조건
* 배열 arr의 크기 : 1,000,000 이하의 자연수
* 배열 arr의 원소의 크기 : 0보다 크거나 같고 9보다 작거나 같은 정수

## 2. 계획
큐(선입선출)를 활용한다~
1/ 반복문으로 [i-1] == [i]이면 arr.shift()해준다.

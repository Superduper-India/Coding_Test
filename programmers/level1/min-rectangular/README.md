# 최소직사각형
## 1. 문제
***
### 원하는 것
모든 명함을 수납할 수 있는 가장 작은 지갑을 만들 때, 지갑의 크기를 return 하도록 solution 함수를 완성

### 조건
* 2차원 배열 sizes는 모든 명함의 가로 길이와 세로 길이를 나타낸다.
* sizes의 원소는 [w, h] 형식입니다.
* w는 명함의 가로 길이를 나타냅니다.
* h는 명함의 세로 길이를 나타냅니다.

## 2. 계획
1/ 가로보다 세로가 크면 서로 위치를 바꾼다.
2/ 가로, 세로의 최대값을 구해서 서로 곱한다.

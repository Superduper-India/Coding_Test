# 짝수와 홀수
## 1. 문제
***
### 원하는 것
정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성

### 조건
* num은 int 범위의 정수
* 0은 짝수

## 2. 계획
1/ num%2 == 0 일 경우, 짝수 ==> "Even" 리턴
2/ num%2 == 1 일 경우, 홀수 ==> "Odd" 리턴

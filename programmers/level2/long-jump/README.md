멀리뛰기에 사용될 칸의 수 n이 주어질 때, 효진이가 끝에 도달하는 방법이 몇 가지인지 알아내, 여기에 1234567를 나눈 나머지를 리턴하는 함수, solution을 완성
// 1234567을 나눈 나머지를 리턴하는 함수? 이해가 안감

### 조건

- 효진이는 한번에 1칸, 또는 2칸을 뛸 수 있다.
- 칸이 총 4개 있을 때, 효진이는
  (1칸, 1칸, 1칸, 1칸)
  (1칸, 2칸, 1칸)
  (1칸, 1칸, 2칸)
  (2칸, 1칸, 1칸)
  (2칸, 2칸)
  의 5가지 방법으로 맨 끝 칸에 도달할 수 있다.

## 2. 계획

1. 가격이 떨어지지 않으면 (즉, 다음 인덱스의 요소가 현재 인덱스의 요소에 비해서 크면)
   => 전체초 - 현재초(index+1)

2. 가격이 떨어지면 (즉, 다음 인덱스의 요소가 현재 인덱스의 요소에 비해서 작으면)
   => 현재초 - 이전초

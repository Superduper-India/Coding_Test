# ❌ 숫자의 표현

## 1. 문제

---

### 원하는 것

자연수 n이 매개변수로 주어질 때, 연속된 자연수들로 n을 표현하는 방법의 수를 return하는 solution를 완성

### 조건

자연수 n을 표현하는 방법은 연속된 자연수들이여야 한다.

## 2. 계획

- 자연수 n을 연속한 자연수들로 표현하는 방법의 수는 n의 약수 중 홀수의 개수와 같다.
- 여기서 포인트는, ⭐️ 모든 홀수(2n+1)은 n과 n+1로 표현이 가능하다는 것이다.
